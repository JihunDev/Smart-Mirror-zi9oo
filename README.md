## Getting Sarted
### Requirements

- Raspberry Pi 3
- USB Mic
- 모니터
- HDMI to HDMI 케이블
- Micro SD Card

### Installation

```Shell
git clone https://github.com/evancohen/smart-mirror.git
sudo apt-get install sox libatlas-base-dev
npm install
```
### Usage
```shell
npm start
```
